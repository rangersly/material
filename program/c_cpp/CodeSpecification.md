## naming

|格式           |说明                    |
|---------------|------------------------|
|全大写         |常量,宏定义             |
|`g_`开头       |全局变量                |
|首字母大写     |结构,类                 |

## code

1. 除特殊需要,嵌套不得超过5层,特别是逻辑部分代码
2. 将函数的定义与声明分离,声明放在头文件(接口函数),文件头部(内部函数)
3. 'C' 显式声明函数的链接域 `extern` 或 `static`
