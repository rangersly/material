# windows 快捷键

### 一、通用操作(几乎每天必用)
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + C` / `Ctrl + X` / `Ctrl + V` | 复制 / 剪切 / 粘贴 | 经典三件套,文件、文本通用 |
| `Ctrl + Z` / `Ctrl + Y` | 撤销 / 重做 | 后悔药,大部分软件支持 |
| `Ctrl + A` | 全选 | 选中当前窗口所有内容 |
| `Ctrl + F` | 查找 | 在当前页面或文档中搜索 |
| `Ctrl + S` | 保存 | 随时保存,防崩溃神器 |
| `Alt + Tab` | 切换窗口 | 按住Alt不放,多次按Tab选择 |
| `Alt + F4` | 关闭当前程序 | 也可用于快速关机(桌面时) |
| `F2` | 重命名 | 选中文件/文件夹后直接改名 |
| `F5` | 刷新 | 刷新桌面或文件夹 |
| `Delete` | 删除到回收站 |  |
| `Ctrl + Shift + T` | 恢复刚刚关闭的浏览器标签页 | 浏览器通用 |

---

### 二、窗口管理(多任务必备)
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Win + D` | 显示桌面 | 再次按恢复窗口 |
| `Win + M` | 最小化所有窗口 | 与Win+D类似,但恢复方式不同 |
| `Win + 左/右箭头` | 窗口分屏 | 将窗口贴靠到屏幕左/右半边 |
| `Win + 上/下箭头` | 窗口最大化/还原/最小化 | 按上最大化,按上再按上还原,按下最小化 |
| `Win + Home` | 最小化除当前窗口外的所有窗口 | 保持专注 |
| `Win + Shift + 左/右箭头` | 将窗口移动到另一台显示器 | 多屏用户必备 |

---

### 三、文件资源管理器高效操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Win + E` | 打开文件资源管理器 | 快速访问磁盘 |
| `Alt + 左/右箭头` | 后退/前进文件夹 | 类似浏览器导航 |
| `Alt + 上箭头` | 返回上一级文件夹 | 比用鼠标点更方便 |
| `Ctrl + N` | 打开新窗口 | 在当前路径再开一个资源管理器 |
| `Ctrl + W` | 关闭当前窗口 |  |
| `Ctrl + Shift + N` | 新建文件夹 | 秒建文件夹 |
| `F11` | 全屏/退出全屏 | 在文件夹或浏览器中可用 |
| `Ctrl + 鼠标滚轮` | 缩放文件图标大小 | 列表/详细信息视图下也有效 |

### 四、文本编辑(办公/码字神器)
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + B` / `I` / `U` | 加粗 / 斜体 / 下划线 | 多数文本编辑器支持 |
| `Ctrl + Shift + 左/右箭头` | 按单词选择文字 | 比按住鼠标精准 |
| `Shift + Home/End` | 选择从光标到行首/行尾 |  |
| `Ctrl + 左/右箭头` | 光标跳转到单词开头 | 快速移动 |
| `Ctrl + Backspace` | 删除前一个单词 | 删除整词,省力 |

### 五、系统功能与工具(效率翻倍)
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Win` | 打开/关闭开始菜单 |  |
| `Win + I` | 打开设置 | 系统设置入口 |
| `Win + R` | 打开运行对话框 | 输入命令快速启动程序(如`cmd`、`regedit`) |
| `Win + Q` 或 `Win + S` | 打开搜索 | 全局搜索文件、应用、设置 |
| `Win + X` | 打开快捷菜单 | 电源用户菜单,含磁盘管理、终端等 |
| `Win + P` | 投影选项 | 复制、扩展屏幕等 |
| `Win + Print Screen` | 截图并自动保存 | 图片保存在`此电脑/图片/屏幕截图` |
| `Win + Shift + S` | 打开截图工具 | 可自定义截图区域(矩形、任意、全屏) |
| `Win + V` | 打开剪贴板历史 | 查看之前复制过的多条内容(需在设置中开启) |
| `Win + .` 或 `Win + ;` | 打开表情符号面板 | 输入颜文字、特殊符号、剪贴板历史 |
| `Ctrl + Shift + Esc` | 打开任务管理器 | 快速查看进程、结束任务 |
| `Win + L` | 锁定电脑 | 离开时立刻锁屏,保护隐私 |

### 六、虚拟桌面(Windows 10/11)
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Win + Tab` | 打开任务视图 | 查看所有桌面和打开的应用 |
| `Win + Ctrl + D` | 新建虚拟桌面 | 快速创建新桌面 |
| `Win + Ctrl + 左/右箭头` | 切换相邻虚拟桌面 | 在不同桌面间跳转 |
| `Win + Ctrl + F4` | 关闭当前虚拟桌面 | 该桌面上的窗口会自动移到其他桌面 |

### 七、其他实用“生僻”快捷键
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Win + 数字键(1~9)` | 打开任务栏上对应位置的程序 | 比如按`Win+1`打开第一个图标,`Win+2`打开第二个,快速启动 |
| `Win + T` | 在任务栏图标间循环切换 | 按回车打开,按右键可展开列表 |
| `Alt + Enter` | 显示所选项目的属性 | 对文件/文件夹按此组合,快速查看属性 |
| `Shift + 右键点击` | 在文件上打开高级上下文菜单 | 可以显示“复制为路径”、“在此处打开PowerShell窗口”等隐藏选项 |
| `Ctrl + Shift + 点击任务栏图标` | 以管理员身份运行该程序 | 直接绕过UAC弹窗确认?实际会触发UAC,但比右键快捷 |
| `Win + Pause/Break` | 打开系统属性 | 查看电脑基本信息、设备管理器入口 |
