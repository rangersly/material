## 录制宏

使用 q[Key] 开始录制宏,再次点击q结束录制
@[Key]	调用宏
@@		重复调用

##文件内移动

+ `0 $` 行首行尾
+ `[n]gg`     移动到第n行
+ `f[c]` 定位到行内字符	`; ,` 定位下一个
+ `c-u` 上半屏
+ `c-d` 下半屏
+ `Shift-[` 跳转到上一个空行,右中括号反之
+ `*` 移动到下一个与当前指针下相同单词的位置
+ `#` 移动到上一个与当前指针下相同单词的位置
+ `` `[a-z] / `[A-Z] ``   跳转到标记位置 大写全局,小写本文件,用m[a-z]设置标记
  + `` ` `` 回到上次跳转的位置

## 文件间跳转

+ `gf` 跳转到文件路径
+ `c-o` 后退一步
+ `c-i` 前进一步

## 插入

+ `I A` 行首行末插入
+ `i a` 光标前光标后插入

## 修改

+ `r` 替换光标下的字符
+ `c` 修改内容

## 删除

+ `x` 删除光标下字符
+ `J` 与下一行合并
+ `di"` 删除引号内包围的内容(不包括引号),同理可替换为其他范围符号,i改a可以包括引号

##	可视模式

- `o` 交换选择区域另一端和光标的位置
- `aw as ap ab aB` 选择一个单词 句子 段落 () {}
- `vi{` 选中一整个大括号内
- `va{` 选中一整个大括号全部

## 补全菜单

1. `c-n``c-p`  上下移动
2. `c-y`  确认
3. `c-e`  退出

## 撤回

- `u` 撤回上一步
- `c-r` 撤回u撤回的内容
