# tmux 现代强大的终端复用工具

## 核心概念

- **层级结构**
  - `会话 -> 窗口 -> 窗格`
  - **会话**:最高层级,包含多个窗口
  - **窗口**:相当于浏览器标签页,占满整个终端
  - **窗格**:窗口内的分割

- **会话操作**
  - `tmux new -s session` 启动新命名会话
  - `Ctrl+B,D` 分离会话
  - `tmux ls` 查看所有会话
  - `tmux a -t session` 恢复会话(不加-t参数就是恢复最近)
  - `Ctrl+B,S` 切换会话

- **窗口操作**
  - 一切以 `Ctrl+B` 为**前置**,下面省略
  - `c` 创建新窗口
  - `n` 切换到下一个窗口
  - `p` 切换到上一个窗口
  - `0-9` 切换到指定窗口
  - `&` 关闭窗口
  - `w` 显示窗口列表交互选择

- **窗格操作**
  - 一切以 `Ctrl+B` 为**前置**,下面省略
  - `%` 垂直分割
  - `"` 水平分割
  - `方向键` 按方向切换窗格
  - `q` 显示窗格编号,按数字切换
  - `Alt+方向键` 调整窗格大小
  - `Space` 循环切换预设布局
  - `x` 关闭窗格
  - `{` 与上一个窗格交换位置
  - `z` 最大化/恢础复 窗格
  
## 配置文件

用户根目录下`.tmux.conf`

- `set -g mouse on` 开启鼠标支持
