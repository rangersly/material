# Win+R 大合集

|命令|作用|
|---|---|
|cmd|打开cmd|
|dxdiag|电脑配置|
|cleanmgr|清理磁盘|
|regedit|注册表|
|msconfig|系统配置|
|mstsc|远程桌面|
|calc|计算器|

# cmd命令大合集

|命令|作用|
|---|---|
|`arp -a`|arp缓存|
|`fsutil behavior query disabledeletenotify`|命令提示符查看系统有无开启TRIM回收指令|
|`fsutil behavior set disabledeletenotify 1`|开启TRIM|
|`fsutil behavior set disabledeletenotify 0`|关闭TRIM|
|`tracert`|显示数据包到达目的主机所经过的路径|
|`ipconfig/all`|显示当前TCP/IP配置的设置值|
|`ipconfig /flushdns`|清空DNS缓存|
|`nslookup`|查询任何一台机器的IP和对应的域名|
|`netstat`|显示活动的TCP连接，计算机侦听的端口，以太网统计信息，IP路由表，IPv4统计信息以及IPv6统计信息|
|`netstat -o`|看连接别人IP|
|`powercfg -h off`|关闭休眠，可节省C盘空间|
|`shutdown /r /fw /t 0`|重启到BIOS|
